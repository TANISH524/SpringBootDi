# Spring Boot Dependency Injection Demo

A beginner project demonstrating core Spring Boot concepts
including Dependency Injection, Beans, and Components.

## Technologies Used
- Spring Boot 3.3.5
- Java 17

## Concepts Covered

### 1. Dependency Injection (DI)
- @Autowired annotation
- Constructor Injection
- Singleton Pattern

### 2. Spring Annotations
- @Component  - Class ko Spring Bean banata hai
- @Autowired  - Dependency inject karta hai
- @Service    - Service layer
- @Repository - Database layer

### 3. Spring Container
- ConfigurableApplicationContext
- getBean() method
- Bean lifecycle
